> 2) I see open type fonts (otf), but no indications of how to use

> even MnSymbol 1.3 can be used with XeTeX with the following

The otf are provided in order to ease usage/installation for non-TeX  
application. For xetex using the xdvipdfmx route you can use the tfm/ 
pfb combination. The xdv2pdf route doesn't currently work with  
neither tfm/pfb nor otf: tfm are not supported by this driver and  
xetex's unicode math package isn't complete yet. But even if it were  
complete, I don't know how it'd get to know the math fontdimens,  
charlists and the 'extensible' information.
