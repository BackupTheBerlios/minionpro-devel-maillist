Sorry my delayed post. The unique certainty I have by now is that the
"--pack" option triggers the Yap's bug. Even creating a new glyph list
with the generate-glyph-list.sh script and then using the "--pack"
option with the just generated list can (unfortunately) trigger the
annoying bug. The only safe way for MiKTeX (and Yap!) users is not to
use the "--pack" option.
