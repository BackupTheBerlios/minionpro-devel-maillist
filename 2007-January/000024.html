Sebastian Schubert schrieb:
> Hi,
> 
> What do you think of printing bold math using semibold not bold?  If I
> have math included in the headlines I like to have it some kind of bold
> as well.  \boldmath uses bold in opposite to semibold of the headline
> which looks strange.  So a bold=eb would solve this problem.  If one
> needs a bolder math symbol (eg in the headlines) there is still eb.
> Unfortunately, bm seems to still use eb.

How about

\documentclass{scrbook}
\usepackage[fullfamily,opticals]{MinionPro}
\addtokomafont{sectioning}{\rmfamily\mathversion{sbold}}
\makeatletter
   \DeclareMathVersion{sbold}
   \SetSymbolFont{operators}{sbold}{T1} {\Mn at Math@Family}{b}{n}
   \SetSymbolFont{letters}  {sbold}{OML}{MinionPro-OsF} 
{b}{\Mn at Math@LetterShape}
   \SetMathAlphabet\mathit  {sbold}{T1} {\Mn at Math@Family}{b}{it}
\makeatother
\SetSymbolFont{MnSyA}       {sbold}{U}  {MnSymbolA}{b}{n}
\SetSymbolFont{MnSyB}       {sbold}{U}  {MnSymbolB}{b}{n}
\SetSymbolFont{MnSyC}       {sbold}{U}  {MnSymbolC}{b}{n}
\SetSymbolFont{MnSyD}       {sbold}{U}  {MnSymbolD}{b}{n}
\SetSymbolFont{largesymbols}{sbold}{OMX}{MnSymbolE}{b}{n}
\SetSymbolFont{symbols}     {sbold}{U}  {MnSymbolF}{b}{n}

\begin{document}
   \chapter{Hallo $a=b$}
   Hallo
\end{document}
