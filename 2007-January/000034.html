Dear Achim

I tried the latest version (only MinionPro.sty in the directory cvs)
and there is no difference between loosequotes option or not.
I am not a specialist in latex package construction but I modified
MinionPro.sty in the following way
\RequirePackage[kerning=true]{microtype}} instead of \RequirePackage{microtype}}
shape    = {n,sc,it} instead of shape = n
and then it seems OK for me.

kerning=true is for microtype 2.0beta 
shape={n,sc,it} is for a bigger space not only in normal (upright) shape.
