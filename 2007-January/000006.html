On Sat, Jan 13, 2007 at 14:41 -0600, Axel E. Retif wrote:
> Is MnSymbol.map a Map or a MixedMap? I had no problems enabling it as  
> a Map file, but now that I'm shifting to version 1.4 from 1.3, I want  
> to do it right.

I would use MixedMap, since MixedMap is meant for fonts whoch exist as
both Type1 and Metafont, whihc is the case for MnSymbol. However, as
long as dvipsPreferOutline is set to true in updmap.cfg, there is no
differnce between Map and MixedMap in default operations. However, if
you use MixedMap you are able to explicitly select the Metafont versions
via 'dvips -Ppk'.
