microtype 2.0 just came out and it still supports \SetExtraKerning.

Anyway, I'd be in favour of adding a compatibility switch. Say, in 
miktex the user installed MnSymbol via the package manager, the

BTW, something with the minionint option seems to be broken,
\documentclass{article}
\usepackage[minionint]{MinionPro}
\begin{document}
\[\iint\]
\end{document}

Compiling this with dvips works flawlessly, with pdftex the integral is 
distorted.
