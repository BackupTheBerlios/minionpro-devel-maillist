Hello

First I would like to thank people who
made MinionPro and MnSymbol packages.
Because in french we have a lot of apostrophes
and because the space seems too small for me
I wanted to use the loosequote option which
gives an error message.
So I use in the preamble of my document the following
\SetExtraKerning
          { encoding = {OT1,T1,LGR,U,OT2,T2A,T2B,T2C,T5,X2},
           family   = {MinionPro-OsF,MinionPro-LF,MinionPro-TOsF,MinionPro-TLF},
          series= {m,b,i,s},
                  shape    = {n,sc,it} }
         { \textquotedblleft = {30,30},  \textquotedblright = {30,30},
           \textquoteleft    = {30,30},  \textquoteright    = {30,30}}
which is not a very elegant solution.

I use microtype v2.0-beta 7, and pdftex 1.40.0-beta-20061017 
which is patched to avoid the error message 
(in pure pdf output) "This can't happen (flushing)".
I plan to compile-install pdftex-1.40.1.

best regards
