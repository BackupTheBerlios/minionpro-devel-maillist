Apparently, you used updmap-sys. Have you run updmap as the user you 
want to use the font before? This would have created something like 
.texmf-var (or similar). The settings in this folder have precedence 
over the system setting. Updating the system settings by running updmap-
sys has therefore no effect on the user settings. Either delete the user 
settings or run updmap for the user.
