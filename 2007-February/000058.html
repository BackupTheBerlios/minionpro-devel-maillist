Hello Michael,

> In the different font versions the non-standard ligatures are assigned
> different encoding slots.

the same is true for /zero.denominator and /zero.numerator and perhaps
other symbols that are not fixed to an encoding slot. (One of my scripts
spotted only the cases where the mapping is ambiguous, e.g.:

> ./analyze map-2000-1000
WARNING: /.notdef => /.notdef /threequartersemdash
WARNING: /T_h => /f_b /T_h
WARNING: /f_b => /f_b /f_h
WARNING: /f_f_j => /f_f_j /f_f_k
WARNING: /f_f_t => /f_f_j /f_f_t
WARNING: /f_h => /f_h /f_j
WARNING: /f_j => /f_j /f_k
WARNING: /f_k => /f_k /f_t
WARNING: /f_t => /f_t /T_h
WARNING: /zero.denominator => /uniF655 /uniF661
WARNING: /zero.numerator => /uniF655 /uniF661

Of course, there could be many more (non-ambiguous) wrong mappings.)
