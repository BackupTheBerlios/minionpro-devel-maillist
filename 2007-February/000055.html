Hello,

for those that use the 'abx' option; I've converted mathabx' 
calligraphic font to pfb (not by tracing, but by metapost+fontforge), 
see http://www.hft.ei.tum.de/mz/mathcm10.pfb

In order to use it, create a map file 'mathabx.map' containing
   mathcm10 mathcm10 < mathcm10.pfb
copy it to localtexmf/fonts/map/dvips/, the pfb goes to 
localtexmf/fonts/type1/public/mathabx

Then run
   mktexlsr
   updmap-sys --enable MixedMap mathabx.map
