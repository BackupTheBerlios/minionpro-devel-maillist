Sorry my ignorance, but Adobe Minion Pro fonts seem to be smaller (by
design) than Times Roman or Palatino at the same point size, since
there is no option like "scaled" in the MinionPro package, what can
one do to change the "base" font size to, say, 13 pt easily? One
solution would be to redefine every LaTeX font command this way:

\makeatletter\renewcommand\normalsize{\@setfontsize\normalsize{24}{48}}\makeatother

But this way one have also change the default baselineskip for each
font size, probably messing other things. Is there any support in the
MinionPro package for this size changing or are there any known
package to support this kind of requirement?

Thanks in advance,
